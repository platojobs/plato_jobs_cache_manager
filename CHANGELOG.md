## 0.0.1

* Initial release of PlatoJobs Cache Manager
* Features include:
  - File caching with automatic management
  - Cache statistics tracking
  - Multiple eviction strategies (LRU, LFU, FIFO)
  - Configurable cache limits and cleanup
  - Support for Android and iOS platforms
  - Concurrent downloads with custom headers
  - Progressive download support
  - File preloading capability
